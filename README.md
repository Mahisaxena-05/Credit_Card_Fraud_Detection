# Credit Card Fraud Detection using Machine Learning

## Project Overview

This project detects fraudulent credit card transactions using Machine Learning. Since fraud datasets are highly imbalanced, I applied **SMOTE (Synthetic Minority Over-sampling Technique)** to balance the training data before building the model.

The final model was trained using **Random Forest Classifier** and evaluated using metrics suitable for imbalanced classification.
